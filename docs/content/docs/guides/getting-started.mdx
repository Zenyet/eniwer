## Overview

External plugins can be loaded into Eniwer via two methods:
- **Pre-init**: Set `window.__thePanelPlugins` before Eniwer initializes
- **Late registration**: Call `window.__thePanelRegisterPlugin()` after Eniwer is ready

## Create a Plugin

Create a JavaScript file that defines and registers your plugin:

```javascript
// my-plugin.js
(function() {
  const plugin = {
    id: 'word-counter',
    name: 'Word Counter',
    description: 'plugin.wordCounter.description',
    icon: '#',

    activate(ctx) {
      this.ctx = ctx;
    },

    deactivate() {
      this.ctx = null;
    },

    // CommandContributor
    getCommands() {
      return [{
        id: 'count-words',
        action: 'countWords',
        label: 'Count Words',
        icon: '#',
        enabled: true,
        order: 200,
      }];
    },

    handleCommand(action, selectedText) {
      if (action !== 'countWords') return false;

      const text = selectedText || this.ctx.getSelectedText();
      const count = text.trim().split(/\s+/).filter(Boolean).length;
      this.ctx.showToast(`Word count: ${count}`);
      return true;
    },
  };

  // Register
  if (window.__thePanelRegisterPlugin) {
    window.__thePanelRegisterPlugin(plugin);
  } else {
    window.__thePanelPlugins = window.__thePanelPlugins || [];
    window.__thePanelPlugins.push(plugin);
  }
})();
```

## Load the Plugin

Inject the script into web pages using a userscript manager (Tampermonkey, Violentmonkey) or another browser extension.

### Tampermonkey Example

```javascript
// ==UserScript==
// @name         Eniwer Word Counter
// @match        *://*/*
// @run-at       document-start
// ==/UserScript==

(function() {
  window.__thePanelPlugins = window.__thePanelPlugins || [];
  window.__thePanelPlugins.push({
    id: 'word-counter',
    name: 'Word Counter',
    activate(ctx) { this.ctx = ctx; },
    deactivate() {},
    getCommands() {
      return [{
        id: 'count-words',
        action: 'countWords',
        label: 'Count Words',
        icon: '#',
        enabled: true,
        order: 200,
      }];
    },
    handleCommand(action) {
      if (action !== 'countWords') return false;
      const text = this.ctx.getSelectedText();
      this.ctx.showToast(`Words: ${text.trim().split(/\s+/).length}`);
      return true;
    },
  });
})();
```

## Plugin States

Users can enable/disable plugins from the settings panel. Plugin states are persisted in the extension config under `pluginStates`:

```json
{
  "pluginStates": {
    "word-counter": true,
    "youtube": false
  }
}
```

When a plugin is disabled:
- Its commands are hidden from the palette
- Its settings section is not rendered
- Its views are inaccessible
- `deactivate()` is called immediately

When re-enabled, `activate()` is called again with a fresh `PluginContext`.

## Using PluginContext

The `ctx` object provides rich capabilities. Here are common patterns:

### Show a Toast

```javascript
ctx.showToast('Operation completed!');
```

### Call AI

```javascript
const result = await ctx.ai.call(
  'Summarize: ' + selectedText,
  'You are a helpful summarizer.'
);
if (result.success) {
  ctx.showToast(result.result);
}
```

### Persist Data

```javascript
// Scoped to your plugin automatically
await ctx.storage.set('lastRun', Date.now());
const lastRun = await ctx.storage.get('lastRun');
```

### Send Background Messages

```javascript
const response = await ctx.sendMessage({ type: 'MY_CUSTOM_ACTION', payload: data });
```

### Check Other Plugins

```javascript
if (ctx.plugins.isEnabled('contextChat')) {
  ctx.plugins.handleCommand('contextChat', 'Hello');
}
```

For full API reference, see [PluginContext](/docs/plugin-api/plugin-context).
